Preguntas Frecuentes

Si ya tengo un sitio web, ¿puedo migrar a su web hosting?

Si tienes acceso a los archivos de tu sitio web existente, simplemente puedes cargarlos a través de un cliente de FTP o desde el Panel de Control del Hosting. Si no tienes una copia de seguridad, deberías poder obtener una copia de ellos de tu proveedor de hosting actual.

¿Qué plan de Web hosting debería elegir?

Si deseas montar un sitio web que se compone de simples páginas estáticas o si se trata de una aplicación web como un blog WordPress, catalogo, carrito de compras, etc., cantidad de cuentas de correos electrónicos que vas a utilizar, Esto te puede dar una idea del espacio en disco que podrás necesitar, el tráfico que prevés para tu sitio web y tus posibilidades económicas (personal o profesional).

¿Qué tan pronto estará lista mi cuenta?

Por lo general las activaciones una vez confirmado el pago se realizan de forma inmediata, esto podría depender también de la cantidad de activaciones pendientes pautadas para el día.

¿Realizan activacion de dominios?

Registramos su dominio nacional o internacional a los mejores precios y quedan a su nombre.
